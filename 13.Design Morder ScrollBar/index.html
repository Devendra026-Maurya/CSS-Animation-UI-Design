HTML stands for HyperText Markup Language. It is the standard language used to create and structure content on the web. HTML tells the browser how to display text, images, links, and other elements on a webpage.

CSS stands for Cascading Style Sheets. It is used to style and design HTML elements. While HTML builds the structure of a webpage, CSS makes it look beautiful — like adding colors, spacing, borders, fonts, layout, animations, and more.

JavaScript (JS) is a programming language used to add interactivity to websites. While HTML builds the structure and CSS styles the page, JavaScript makes it dynamic (like button clicks, pop-ups, sliders, etc.).
