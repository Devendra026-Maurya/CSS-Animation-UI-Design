# CSS Animation UI Design Projects

A collection of beautiful and responsive **CSS Animation UI Design Projects** created using **HTML5** and **CSS3**. These projects are perfect for beginners and intermediate frontend developers who want to improve their skills in animations, transitions, hover effects, and responsive layouts.

## 🚀 Projects Included

1. Responsive Profile Card
2. Glowing Buttons
3. Credit Card UI
4. Flipping Card UI
5. Simple Website
6. Animation Effect Background
7. Accordion Example
8. Add Background Image to Text
9. Text Card Design
10. Image Background Flip
11. Image Hover Effect
12. Dynamic Loading Animation
13. Modern Scrollbar Design
14. Button Design UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Media Queries
- CSS Animations
- CSS Transitions

📸 Preview
Each folder contains standalone project files that you can open directly in your browser to see the animation effects in action.

Clone the repository:
git clone https://github.com/Devendra026-Maurya/CSS-Animation-UI-Design-Projects.git
